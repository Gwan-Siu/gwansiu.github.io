# 2. What's the Machine Learning?
## 2.1 The definition of machine Learning
机器学习方法是**计算机利用已有的经验(观测数据)**，解算出某个模型(规律)，并利用该模型来预测未来的一种方法。该定义出发点是让计算机模拟人脑的学习过程，将观察的经验归纳总结出某种规律，并利用该规律观察自然，并改造自然。
